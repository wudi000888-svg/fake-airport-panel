# fake-ui v2.3.0

v2.3.0 是一次账号安全和响应式后台体验更新。它在 v2.2.0 商业运营仪表盘基础上，补齐已登录用户修改密码、管理员修改密码、移动端抽屉导航、桌面侧栏收起和窄屏布局稳定性。

## Updates

- 普通用户可在用户中心修改自己的面板登录密码，后端要求校验当前密码。
- 管理员可在系统设置页修改当前管理员登录密码。
- 密码更新继续使用 PBKDF2-SHA256 存储，不在前端或日志中保存明文密码。
- 登录后的账号入口改为头像菜单，提供个人资料、订阅管理和退出登录。
- 桌面侧栏底部移除退出按钮，改为“收起/展开”控制。
- 桌面侧栏可收起为 88px 图标栏，主内容区域会随侧栏宽度重新计算。
- 移动端移除底部导航栏，改为顶部菜单按钮和侧边抽屉。
- 修复窗口缩小后“收起”按钮失效的问题：小屏下点击会关闭抽屉，桌面下点击才折叠侧栏。
- 修复 319px 等窄屏横向溢出，移动端顶部栏、账号菜单、侧栏和主内容不会撑破视口。
- 前端结构测试覆盖移动抽屉、账号菜单、侧栏折叠、无底部导航和自助改密入口。

## Upgrade Notes

从 v2.2.0 可直接滚动升级到 v2.3.0。此版本不包含数据库 schema 迁移。

运行态 `.env`、`data/`、`generated/`、证书、用户数据、订阅 token、真实域名、用户密码和私钥必须继续保留在服务器，不应提交到 Git。
